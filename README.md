\# Expense Tracker



A simple Java-based Expense Tracker application that helps users record, manage, and monitor their daily expenses efficiently.



\## Features



\- Add new expenses with details (amount, category, date)

\- View all recorded expenses

\- Categorize expenses (e.g., Food, Transport, Bills)

\- Track spending over time

\- Simple console-based interface (or JavaFX if applicable)



\## Technologies Used



\- Java

\- File Handling / Collections (depending on implementation)

\- OOP Principles



\## How to Run



\### 1. Compile the program

```bash

javac "Expense tracker.java"



2\. Run the program

java ExpenseTracker



⚠️ Make sure the class name inside the file matches the file name (recommended: ExpenseTracker.java)
